#+TITLE: eldoc-box

Display Eglot hover documentation on demand in an Emacs child frame.

This fork is intentionally small: it does not provide automatic hover modes,
Eldoc buffer integration, Markdown prettifier hooks, or cleanup timers. Bind one
command to a key and press it once to show documentation. Move point or press it
again to hide the popup.

* Install

Put =eldoc-box.el= on your =load-path= and require it:

#+BEGIN_SRC emacs-lisp
(require 'eldoc-box)
#+END_SRC

* Usage

Bind the toggle command in buffers managed by Eglot:

#+BEGIN_SRC emacs-lisp
(with-eval-after-load 'eglot
  (define-key eglot-mode-map (kbd "C-c h") #'eldoc-box-toggle-help-at-point))
#+END_SRC

Available commands:

- =eldoc-box-toggle-help-at-point= :: Toggle Eglot hover documentation at point.
- =eldoc-box-eglot-help-at-point= :: Request and display Eglot hover
  documentation.
- =eldoc-box-quit-frame= :: Close the documentation child frame.

* Customization

- =eldoc-box-border= :: Border face for the child frame.
- =eldoc-box-body= :: Body face for the child frame.
- =eldoc-box-bottom-padding= :: Extra bottom padding, in pixels.
- =eldoc-box-buffer-hook= :: Hook run in the popup buffer before display.
- =eldoc-box-max-pixel-width= :: Maximum popup width.
- =eldoc-box-max-pixel-height= :: Maximum popup height.
- =eldoc-box-position-function= :: Function used to position the popup.
- =eldoc-box-position-offset= :: Pixel offset added to the default position.
